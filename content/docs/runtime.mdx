# Elide Runtime

Use Elide to run JavaScript, Python, Ruby, and more in a single, blazingly fast polyglot runtime.

## What is the Elide Runtime?

The Elide Runtime is a **polyglot execution environment** that allows you to write applications in multiple programming languages that seamlessly interoperate within a single process. Unlike traditional approaches that run separate language runtimes and communicate via serialization, Elide provides a unified runtime powered by GraalVM where languages can directly call each other's functions, share data structures, and work together efficiently.

## Why Elide Excels

### True Polyglot Architecture

- **One Process**: No subprocess orchestration or IPC overhead
- **One Interpreter**: Unified GraalVM Truffle-based execution
- **Zero Serialization**: Share objects and functions directly between languages
- **Seamless Interop**: Import Python from JavaScript, Ruby from Python, etc.

### Performance & Efficiency

- **JIT Compilation**: Fast execution after warmup with GraalVM's optimizing compiler
- **Native Image**: Compile to standalone native binaries with instant startup
- **Low Memory Footprint**: Shared runtime reduces memory overhead
- **World-Class Benchmarks**: Independently benchmarked by TechEmpower

### Modern Standards

- **Web APIs**: Fetch, Crypto, Encoding, URL, and more
- **Node.js Compatibility**: Extensive Node API support
- **WinterTC Conformance**: Aligned with WinterCG standards
- **Built-in Tooling**: Includes oxc, orogene, uv, and Pkl

### Developer Experience

- **Interactive REPL**: Test code instantly in any language
- **Hot Reload**: Watch mode for rapid development
- **Debugging Support**: Step through code across language boundaries
- **Rich Diagnostics**: Clear error messages and stack traces

## CLI Commands

Elide provides a comprehensive command-line interface for running scripts, managing projects, building native binaries, and more.

### Quick Start

The simplest way to run code with Elide:

```shell
$ elide app.js
$ elide script.py
$ elide program.rb
```

Elide automatically detects the language from the file extension (`.js`, `.ts`, `.py`, `.rb`, `.kts`, etc.).

### Runtime Commands

**`elide run [file]`** - Run a script file or code snippet

```shell
$ elide run app.js
$ elide run --code "console.log('Hello!')"
$ elide run script.py
```

Execute scripts in any supported language. The runtime detects the language automatically or you can specify it explicitly.

**`elide repl`** - Start an interactive REPL session

```shell
$ elide repl
$ elide repl --python
```

Opens an interactive read-eval-print-loop for testing code. If no language is specified, JavaScript is used by default.

**`elide serve [file]`** - Run a server application

```shell
$ elide serve server.js
```

Alias to `run` specifically for server applications. Blocks until the server exits.

**`elide test`** - Run tests

```shell
$ elide test
$ elide test pattern
```

Execute test suites for your project.

<div class="border-t border-border/40 my-12"></div>

### Project Management

**`elide project`** - Manage Elide projects

```shell
$ elide project
```

Manage projects defined by `elide.pkl` or other project manifests.

**`elide init`** - Initialize a new project

```shell
$ elide init
```

Create a new Elide project with default configuration files.

**`elide install`** / **`elide i`** - Install dependencies

```shell
$ elide install
$ elide i
```

Resolve and install all dependencies for the current project. Dependencies are installed for all declared ecosystems.

**`elide add`** - Add dependencies

```shell
$ elide add package-name
$ elide add org.example:library:1.0.0
```

Add one or more dependencies to your project and install them. Accepts dependency coordinates in various formats.

**`elide build`** - Build project

```shell
$ elide build
$ elide build --native
```

Build the project or run a script mapped to 'build' in `elide.pkl` or `package.json`.

<div class="border-t border-border/40 my-12"></div>

**`elide dev`** - Development mode

```shell
$ elide dev
```

Run an Elide project in development mode with hot reload and debugging features.

<div class="border-t border-border/40 my-12"></div>

### JVM Toolchain

Elide includes a full JVM toolchain for Java and Kotlin development:

**`elide javac`** - Java compiler

```shell
$ elide javac -- [JAVAC_OPTIONS] [SOURCES...]
```

The Java compiler, supporting up to JDK 24.

**`elide kotlinc`** - Kotlin compiler

```shell
$ elide kotlinc -- [KOTLINC_OPTIONS] [SOURCES...]
```

The Kotlin compiler for compiling Kotlin programs.

**`elide jar`** - JAR builder

```shell
$ elide jar -- [OPTIONS] [FILES...]
```

Builds and manages Java Archive files (JARs).

**`elide javadoc`** - Documentation generator

```shell
$ elide javadoc -- [OPTIONS] [SOURCES...]
```

Renders Java code into documentation.

**`elide native-image`** - Native compiler

```shell
$ elide native-image [OPTIONS]
```

GraalVM Native Image compiler for building standalone native binaries.

**`elide jib`** - Container builder

```shell
$ elide jib [OPTIONS]
```

Build container images from JVM or native applications without Docker.

### Configuration & Tooling

**`elide pkl`** - Pkl configuration

```shell
$ elide pkl [command]
```

Run the <a href="/docs/pkl.html" class="text-primary hover:text-primary/80 transition-colors border-b border-primary/40 hover:border-primary/60" style="text-decoration: none;">Pkl</a> command-line tools for configuration management.

**`elide secrets`** - Secrets management

```shell
$ elide secrets
```

Manage secrets for the current application.

**`elide lsp`** - Language Server Protocol

```shell
$ elide lsp
```

Run an LSP instance for IDE integration and code intelligence.

**`elide mcp`** - Model Context Protocol

```shell
$ elide mcp
```

Run an MCP instance for AI integration.

### Information & Help

**`elide info`** - Show system information

```shell
$ elide info
```

Display information about your Elide installation, including version, supported languages, and environment details.

**`elide which`** - Find tool binaries

```shell
$ elide which [tool]
```

Print the path to a tool binary.

**`elide help`** / **`elide docs`** - Show documentation

```shell
$ elide help
$ elide docs run
```

Display help text for any command or search Elide's documentation.

**`elide bug`** / **`elide issue`** / **`elide feature`** - Report issues

```shell
$ elide bug
$ elide issue
$ elide feature
```

Open links to GitHub to report bugs, issues, or request features.

**`elide discord`** - Join Discord

```shell
$ elide discord
```

Open or show a link to join the Elide Discord community server.

**`elide completions`** - Shell completions

```shell
$ elide completions
```

Generate bash/zsh completion scripts for enhanced shell integration.

**`elide --version`** - Show version

```shell
$ elide --version
```

Print the current Elide version and exit.
