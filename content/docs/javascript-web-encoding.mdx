# Encoding API

Elide supports the standard JavaScript types `TextEncoder` and `TextDecoder`. These classes can be used to obtain byte-arrays of strings in different encodings, or to obtain strings from byte arrays with different encodings.

## Supported Encodings

Elide supports several encodings via the `TextEncoder` and `TextDecoder` constructors:

| Support | Charset | Encoding (Label) | Notes |
|---------|---------|------------------|-------|
| 游릭 Supported | `ISO_8859_1` | `iso-8859-1` | |
| 游릭 Supported | `ASCII` | `ascii` | |
| 游릭 Supported | `UTF_8` | `utf-8` | Default encoding |
| 游댮 Not supported yet | `UTF_16` | `utf-16` | |
| 游댮 Not supported yet | `UTF_32` | `utf-32` | |
| 游댮 Not supported yet | (Other charsets) | | |

## TextEncoder

```javascript
// create an encoder; by default, it uses utf-8
const encoder = new TextEncoder();
encoder.encoding         // returns 'utf-8'
encoder.encode("hello")  // returns `Uint8Array([...])`
```

The `TextEncoder` class behaves very similarly to how it behaves in a browser, or under Node.js; the only difference is that Elide supports additional encodings, on top of the default, which is `utf-8`.

Read more about <a href="https://developer.mozilla.org/en-US/docs/Web/API/TextEncoder" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">TextEncoder on MDN</a>

## TextDecoder

```javascript
// create a decoder; by default, it uses utf-8
const decoder = new TextDecoder();
decoder.encoding         // returns 'utf-8'
const input = new Uint8Array([104, 101, 108, 108, 111]);
decoder.decode(input)  // returns 'hello'
```

The `TextDecoder` class behaves identically to how it behaves in a browser, or under Node.js.

Read more about <a href="https://developer.mozilla.org/en-US/docs/Web/API/TextDecoder" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">TextDecoder on MDN</a>
