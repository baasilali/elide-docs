# Encoding API

Elide provides compatibility with the Web Encoding API for text encoding and decoding operations.

## TextEncoder

The `TextEncoder` interface takes a stream of code points as input and produces a stream of UTF-8 bytes.

## TextDecoder

The `TextDecoder` interface takes a stream of bytes as input and produces a stream of code points.
