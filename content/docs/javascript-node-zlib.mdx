# Zlib

API support and documentation for the `node:zlib` module.

**Module:** `node:zlib`  
**Support:** Alpha  
**Docs:** <a href="https://nodejs.org/api/zlib.html" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Node.js Zlib Docs</a>

## Import

```javascript
// ESM
import zlib from "node:zlib"
```

```javascript
// CJS
const zlib = require("node:zlib")
```
