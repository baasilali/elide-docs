# SQLite in JavaScript

**Module:** `elide:sqlite`  
**Support:** Beta  
**Docs:** <a href="https://bun.sh/docs/api/sqlite" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Bun SQLite Docs</a>

Elide embeds native support for <a href="https://sqlite.org" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">SQLite3</a>, with bindings into JavaScript. The API provided is identical to <a href="https://bun.sh/docs/api/sqlite" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Bun's SQLite bindings</a>.

## Import

```javascript
// ESM
import { Database } from "elide:sqlite"
```

```javascript
// CJS
const { Database } = require("elide:sqlite")
```

## Example

```javascript
import { Database } from "elide:sqlite"
const db = new Database()
db.exec("CREATE TABLE test (id INTEGER PRIMARY KEY, name TEXT);")
db.exec("INSERT INTO test (name) VALUES ('foo');")
db.query("SELECT * FROM test;").get()["name"]
// 'foo'
```
