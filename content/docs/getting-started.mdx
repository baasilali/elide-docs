# Getting Started

Quick guide to installing and running your first Elide application.

## Prerequisites

Before you begin, make sure you have:

- Java 11 or higher
- Basic knowledge of JavaScript/TypeScript or Python

## Installation

Install Elide using your preferred method:

```bash
# macOS/Linux
curl -fsSL https://elide.dev/install.sh | bash

# Windows
iwr https://elide.dev/install.ps1 -useb | iex
```

## Your First Application

Create a simple "Hello World" application:

```javascript
// hello.js
console.log("Hello from Elide!")
```

Run it:

```bash
elide run hello.js
```
