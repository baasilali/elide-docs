# Quick Start Guide

Get up and running with Elide in less than 5 minutes. This guide will walk you through creating your first application.

## Prerequisites

Before you begin, make sure you have Elide installed. If not, see the [Installation Guide](/docs/installation).

```bash
# Verify installation
elide --version
```

## Your First Application

Let's create a simple HTTP server that responds to requests.

### Step 1: Create a Project Directory

```bash
mkdir my-elide-app
cd my-elide-app
```

### Step 2: Create server.js

Create a file called `server.js`:

```javascript
// server.js - A simple HTTP server
const app = Elide.http

// Homepage route
app.router.handle("GET", "/", (request, response) => {
  response.send(200, "Welcome to Elide!")
})

// API endpoint
app.router.handle("GET", "/api/hello", (request, response) => {
  response.json({
    message: "Hello from Elide!",
    timestamp: Date.now(),
    version: "1.0.0"
  })
})

// Echo endpoint
app.router.handle("POST", "/api/echo", (request, response) => {
  const body = request.body
  response.json({
    echoed: body,
    length: body.length
  })
})

console.log("ðŸš€ Server running on http://localhost:8080")
```

## Example 2: TypeScript Application

Elide has first-class TypeScript support with built-in compilation.

### Create app.ts

```typescript
// app.ts - TypeScript with type safety
interface User {
  id: number
  name: string
  email: string
}

interface ApiResponse<T> {
  success: boolean
  data?: T
  error?: string
}

const app = Elide.http

// In-memory user store
const users: Map<number, User> = new Map([
  [1, { id: 1, name: "Alice", email: "alice@example.com" }],
  [2, { id: 2, name: "Bob", email: "bob@example.com" }]
])

// Get all users
app.router.handle("GET", "/api/users", (request, response) => {
  const userList = Array.from(users.values())
  const result: ApiResponse<User[]> = {
    success: true,
    data: userList
  }
  response.json(result)
})

// Get user by ID
app.router.handle("GET", "/api/users/:id", (request, response) => {
  const id = parseInt(request.params.id)
  const user = users.get(id)
  
  if (user) {
    const result: ApiResponse<User> = {
      success: true,
      data: user
    }
    response.json(result)
  } else {
    const result: ApiResponse<never> = {
      success: false,
      error: "User not found"
    }
    response.status(404).json(result)
  }
})

console.log("âœ¨ TypeScript server ready!")
```

### Run TypeScript

```bash
# Elide compiles TypeScript automatically
elide run app.ts
```

<Alert>
  <strong>No build step required!</strong> Elide compiles TypeScript on the fly during development and ahead-of-time for production builds.
</Alert>

## Example 3: Polyglot Application

One of Elide's superpowers is running multiple languages together.

### Create polyglot-app.js

```javascript
// polyglot-app.js - JavaScript calling Python
const app = Elide.http

// Import Python function
const { analyze_sentiment } = await import('./sentiment.py')

app.router.handle("POST", "/api/sentiment", async (request, response) => {
  const { text } = request.body
  
  // Call Python function from JavaScript!
  const sentiment = await analyze_sentiment(text)
  
  response.json({
    text,
    sentiment,
    analyzed_at: new Date().toISOString()
  })
})

console.log("ðŸ”€ Polyglot server running!")
```

### Create sentiment.py

```python
# sentiment.py - Python sentiment analysis

def analyze_sentiment(text: str) -> dict:
    """Analyze sentiment of text"""
    # Simple sentiment analysis
    positive_words = ['good', 'great', 'awesome', 'excellent', 'love']
    negative_words = ['bad', 'terrible', 'awful', 'hate', 'poor']
    
    words = text.lower().split()
    
    positive_count = sum(1 for word in words if word in positive_words)
    negative_count = sum(1 for word in words if word in negative_words)
    
    if positive_count > negative_count:
        sentiment = 'positive'
        score = positive_count / len(words)
    elif negative_count > positive_count:
        sentiment = 'negative'
        score = negative_count / len(words)
    else:
        sentiment = 'neutral'
        score = 0.0
    
    return {
        'sentiment': sentiment,
        'score': score,
        'positive_words': positive_count,
        'negative_words': negative_count
    }
```

### Run Polyglot App

```bash
elide run polyglot-app.js
```

Test it:

```bash
curl -X POST http://localhost:8080/api/sentiment \
  -H "Content-Type: application/json" \
  -d '{"text": "This is awesome and great!"}'
```

## Configuration with elide.pkl

Create an `elide.pkl` file for project configuration:

```pkl
// elide.pkl - Project configuration
app {
  name = "my-elide-app"
  version = "1.0.0"
  entry = "src/main.ts"
}

runtime {
  languages = ["javascript", "typescript", "python"]
  optimize = true
}

server {
  port = 8080
  host = "0.0.0.0"
}

build {
  target = "native"
  output = "dist/"
}
```

Then simply run:

```bash
elide run    # Uses config automatically
elide build  # Builds according to config
```

## Common Commands

| Command | Description |
|---------|-------------|
| `elide run file.js` | Run JavaScript file in JIT mode |
| `elide run file.ts` | Run TypeScript (auto-compiled) |
| `elide run file.py` | Run Python script |
| `elide build --native` | Compile to native binary |
| `elide build --jar` | Create JAR file |
| `elide serve dist/` | Serve static files |
| `elide info` | Show system information |
| `elide --version` | Show Elide version |

## Performance Tips

### Development Mode (JIT)
- Fast iteration
- Code recompilation
- Full debugging support
- Higher memory usage

```bash
elide run --mode=dev server.js
```

### Production Mode (Native)
- Compile once, run forever
- Instant startup
- Minimal memory
- Maximum throughput

```bash
elide build --native --optimize server.js
```

## Troubleshooting

### Server Won't Start

```bash
# Check if port is in use
lsof -i :8080

# Use a different port
PORT=3000 elide run server.js
```

### Native Build Fails

```bash
# Make sure you have GraalVM native-image
elide info

# Install if needed (macOS)
brew install graalvm/tap/graalvm-ce-java21

# Verify
native-image --version
```

### TypeScript Errors

```bash
# Enable verbose logging
elide run --verbose app.ts

# Check TypeScript version
elide info typescript
```
