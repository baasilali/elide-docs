# Database API

Elide provides database access through standard APIs and ORM integration.

## Supported Databases

- **PostgreSQL**
- **MySQL/MariaDB**
- **SQLite**
- **MongoDB**
- **Redis**

## SQL Database Access

### Using SQL

```javascript
// Connect to database
const db = await Elide.db.connect('postgresql://localhost/mydb')

// Execute query
const users = await db.query('SELECT * FROM users WHERE age > $1', [18])

// Parameterized queries
const user = await db.queryOne(
  'SELECT * FROM users WHERE id = $1',
  [userId]
)

// Insert
await db.execute(
  'INSERT INTO users (name, email) VALUES ($1, $2)',
  [name, email]
)

// Transactions
await db.transaction(async (tx) => {
  await tx.execute('INSERT INTO accounts (balance) VALUES ($1)', [1000])
  await tx.execute('INSERT INTO audit_log (action) VALUES ($1)', ['created'])
})
```

## ORM Integration

### Entity Definition

```typescript
import { Entity, Column, PrimaryKey } from '@elide/orm'

@Entity('users')
class User {
  @PrimaryKey()
  id: number
  
  @Column()
  name: string
  
  @Column({ unique: true })
  email: string
  
  @Column({ default: () => new Date() })
  createdAt: Date
}
```

### CRUD Operations

```typescript
// Create
const user = new User()
user.name = 'Alice'
user.email = 'alice@example.com'
await user.save()

// Read
const user = await User.findById(123)
const users = await User.findAll()
const active = await User.find({ active: true })

// Update
user.name = 'Alice Smith'
await user.save()

// Delete
await user.delete()
```
