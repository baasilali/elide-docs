# Polyglot Interop

Call between languages seamlessly in Elide

This exercise is part of the Polyglot 101 intro guide.

One of Elide's most powerful features is the ability to mix multiple languages in one application. You can call Python from JavaScript, JavaScript from Python, and share data between them.

## Before you start

Before running the code below, you'll need to make sure the following is true:

- Elide is installed
- The `elide` binary is executable
- The `elide` binary is on your `PATH`, or otherwise referenced in absolute form

## Calling Python from JavaScript

You can import and use Python modules directly in JavaScript:

```javascript
// Import Python's math module
const math = await Polyglot.import('math')

// Use Python functions
const result = math.sqrt(16)
console.log('Square root:', result)  // 4.0

// Use Python's datetime
const datetime = await Polyglot.import('datetime')
const now = datetime.datetime.now()
console.log('Current time:', now)
```

## Calling JavaScript from Python

Similarly, you can call JavaScript from Python:

```python
import polyglot

# Evaluate JavaScript code
result = polyglot.eval(language='js', string='[1, 2, 3].map(x => x * 2)')
print(result)  # [2, 4, 6]

# Call JavaScript functions
js_sum = polyglot.eval(language='js', string='(a, b) => a + b')
total = js_sum(5, 3)
print('Sum:', total)  # 8
```

## Sharing Data

Data automatically converts between languages:

```javascript
// JavaScript creates data
const data = {
  name: 'Alice',
  scores: [85, 90, 95],
  active: true
}

// Pass to Python
const statistics = await Polyglot.import('statistics')
const mean = statistics.mean(data.scores)
console.log('Average score:', mean)  // 90
```

## Real-World Example

Combine JavaScript for web APIs and Python for data processing:

```javascript
// fetch-and-process.js

// Use JavaScript's fetch API
const response = await fetch('https://api.example.com/data')
const data = await response.json()

// Use Python's statistics module
const stats = await Polyglot.import('statistics')

// Calculate statistics on the data
const values = data.map(item => item.value)
const mean = stats.mean(values)
const median = stats.median(values)
const stdev = stats.stdev(values)

console.log({
  mean,
  median,
  stdev
})
```

## Data Type Conversion

Elide automatically converts data types between languages:

| JavaScript | Python |
|------------|--------|
| number | int/float |
| string | str |
| boolean | bool |
| null | None |
| Array | list |
| Object | dict |

## Use Cases

Polyglot interop is perfect for:

- **Data Analysis**: Use Python's scientific libraries with JavaScript's async APIs
- **Web Scraping**: Fetch with JavaScript, parse with Python
- **Machine Learning**: Train models in Python, integrate in JavaScript apps
- **System Scripts**: Combine shell utilities with high-level logic

## Simple Example: Temperature Converter

```javascript
// Use Python for the conversion logic
const converter = await Polyglot.eval('python', `
def celsius_to_fahrenheit(c):
    return (c * 9/5) + 32

def fahrenheit_to_celsius(f):
    return (f - 32) * 5/9

converter
`)

// Call from JavaScript
const tempC = 25
const tempF = converter.celsius_to_fahrenheit(tempC)
console.log(`${tempC}°C = ${tempF}°F`)
```
