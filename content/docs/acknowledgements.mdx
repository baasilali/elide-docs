# Acknowledgements

Elide stands on the shoulders of giants. We're grateful to these projects and communities.

## Core Technologies

### GraalVM

Elide is built on [GraalVM](https://www.graalvm.org/), the high-performance polyglot runtime.

- **Project**: GraalVM
- **Organization**: Oracle Labs
- **License**: GPL v2 with Classpath Exception

### Truffle Framework

Our language engines use the Truffle framework for high-performance language implementation.

- **Project**: Truffle
- **Part of**: GraalVM
- **Use**: Language implementation framework

## Language Implementations

### GraalJS

JavaScript and TypeScript support powered by GraalJS.

- **Project**: GraalJS
- **Implements**: ECMAScript 2024+
- **Performance**: State-of-the-art JIT compilation

### GraalPy

Python support via GraalPy.

- **Project**: GraalPy
- **Implements**: Python 3.11+
- **Compatibility**: CPython-compatible

### TruffleRuby

Ruby support through TruffleRuby.

- **Project**: TruffleRuby
- **Implements**: Ruby 3.2+
- **Performance**: Faster than MRI
