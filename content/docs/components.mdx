# Components

Elide applications are built from modular, composable components.

## Core Components

### Runtime Engine
- **Polyglot Execution**: Run multiple languages in one process
- **JIT Compilation**: Dynamic optimization during runtime
- **AOT Compilation**: Ahead-of-time native compilation
- **Memory Management**: Advanced garbage collection

### Module System
- **ES Modules**: Modern JavaScript module system
- **CommonJS**: Node.js compatibility
- **Dynamic Import**: Lazy loading of modules
- **Polyglot Imports**: Import across languages

### HTTP Server
- **High Performance**: 800K+ requests/second
- **Netty-Based**: Battle-tested networking
- **HTTP/2 & HTTP/3**: Modern protocol support
- **WebSockets**: Real-time communication

## Application Components

### Controllers

Handle HTTP requests:

```typescript
@Controller('/api')
class UserController {
  @Get('/users')
  async listUsers() {
    return await userService.findAll()
  }
  
  @Post('/users')
  async createUser(@Body user: User) {
    return await userService.create(user)
  }
}
```

### Services

Business logic layer:

```typescript
@Service()
class UserService {
  constructor(private db: Database) {}
  
  async findAll(): Promise<User[]> {
    return await this.db.query('SELECT * FROM users')
  }
  
  async create(user: User): Promise<User> {
    return await this.db.insert('users', user)
  }
}
```

### Middleware

Request/response processors:

```typescript
function authMiddleware(req, res, next) {
  if (!req.headers.authorization) {
    return res.status(401).json({ error: 'Unauthorized' })
  }
  next()
}
```

### Filters

Global request/response filters:

```typescript
@Filter()
class LoggingFilter {
  async filter(req: Request, res: Response, next: Function) {
    console.log(`${req.method} ${req.path}`)
    await next()
  }
}
```

## Component Lifecycle

```
Initialize → Configure → Start → Run → Shutdown
```

Each component follows this lifecycle:

1. **Initialize**: Component is created
2. **Configure**: Configuration is applied
3. **Start**: Component begins operation
4. **Run**: Component performs its function
5. **Shutdown**: Component cleans up resources

## Dependency Injection

Components can depend on other components:

```typescript
@Injectable()
class EmailService {
  send(to: string, subject: string, body: string) {
    // Send email
  }
}

@Service()
class UserService {
  constructor(
    private db: Database,
    private email: EmailService  // Injected automatically
  ) {}
  
  async register(user: User) {
    const created = await this.db.insert('users', user)
    await this.email.send(user.email, 'Welcome', 'Thanks for signing up!')
    return created
  }
}
```
