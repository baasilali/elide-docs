## Contributing

We welcome contributions! Here's how you can help:

### Ways to Contribute

1. **Code Contributions**
   - Bug fixes
   - New features
   - Performance improvements
   - Documentation

2. **Community Support**
   - Answer questions
   - Help with issues
   - Write tutorials
   - Share your projects

3. **Testing**
   - Report bugs
   - Test beta releases
   - Provide feedback

### Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

See our [Contributing Guide](https://github.com/elide-dev/elide/blob/main/CONTRIBUTING.md) for details.
