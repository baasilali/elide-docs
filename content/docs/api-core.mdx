# Core API Reference

The Elide framework provides modular components for building applications. Choose only what you need.

## Framework Modules

| Module | Platform | Description |
|--------|----------|-------------|
| **core** | All | Essential utilities, type system, collections |
| **base** | JVM, JS, Native | Extended APIs not yet available on all platforms |
| **server** | JVM, Native | HTTP server, routing, middleware |
| **ssr** | JVM, JS | Server-side rendering annotations and logic |
| **graalvm** | JVM, Native | GraalVM integration and JavaScript support |
| **test** | All | Testing utilities and assertions |

## Core Module

### Using Core

```kotlin
// Kotlin
import elide.core.*

// Java
import tools.elide.core.*;
```

```javascript
// JavaScript/TypeScript
import { ... } from '@elide/core'
```

### Collections

Efficient, immutable data structures:

```kotlin
// Persistent list
val list = persistentListOf(1, 2, 3)
val updated = list.add(4) // Original unchanged

// Persistent map
val map = persistentMapOf("key" to "value")
val updated = map.put("key2", "value2")

// Persistent set
val set = persistentSetOf(1, 2, 3)
```

### Type System

```kotlin
// Sealed types for result handling
sealed class Result<T> {
  data class Success<T>(val value: T) : Result<T>()
  data class Error<T>(val error: String) : Result<T>()
}

fun fetchData(): Result<Data> {
  return try {
    Result.Success(loadData())
  } catch (e: Exception) {
    Result.Error(e.message)
  }
}
```

## Server Module

### Annotations

```kotlin
import elide.server.annotations.*

@Page("/")
class HomePage {
  @Get
  fun index(): Response {
    return Response.ok("Hello!")
  }
}

@API("/api")
class UserAPI {
  @Get("/users")
  fun listUsers(): List<User> {
    return userService.findAll()
  }
  
  @Post("/users")
  fun createUser(@Body user: User): User {
    return userService.create(user)
  }
}
```

### Controllers

```kotlin
@Controller
class UserController(
  private val userService: UserService
) {
  @Get("/users/{id}")
  fun getUser(@Path id: String): User {
    return userService.findById(id)
  }
  
  @Put("/users/{id}")
  fun updateUser(
    @Path id: String,
    @Body user: User
  ): User {
    return userService.update(id, user)
  }
}
```

## SSR Module

### Server-Side Rendering

```kotlin
import elide.ssr.*

@SSR
class ProfilePage {
  @Render
  fun render(user: User): String {
    return """
      <!DOCTYPE html>
      <html>
        <head><title>${user.name}</title></head>
        <body>
          <h1>${user.name}</h1>
          <p>${user.email}</p>
        </body>
      </html>
    """
  }
}
```

## GraalVM Module

### Polyglot Integration

```kotlin
import elide.runtime.gvm.*

// Execute JavaScript
val context = createContext()
val result = context.eval("js", "1 + 2")

// Call JavaScript from Kotlin
val jsFunction = context.getBindings("js")
  .getMember("myFunction")

val result = jsFunction.execute(arg1, arg2)
```

## Test Module

### Testing Utilities

```kotlin
import elide.test.*

class UserServiceTest : ElideTest() {
  @Test
  fun `should create user`() {
    val user = User(name = "Alice")
    val created = userService.create(user)
    
    assertThat(created.id).isNotNull()
    assertThat(created.name).isEqualTo("Alice")
  }
  
  @Test
  suspend fun `should fetch user async`() {
    val user = userService.fetchAsync(123)
    assertThat(user.id).isEqualTo(123)
  }
}
```

## Using Framework Modules

### Gradle

```kotlin
dependencies {
  // Core (always needed)
  implementation("dev.elide:elide-core")
  
  // Base (optional)
  implementation("dev.elide:elide-base")
  
  // Server (for HTTP)
  implementation("dev.elide:elide-server")
  
  // SSR (for rendering)
  implementation("dev.elide:elide-ssr")
  
  // GraalVM (for polyglot)
  implementation("dev.elide:elide-graalvm")
  
  // Test (for testing)
  testImplementation("dev.elide:elide-test")
}
```

### Maven

```xml
<dependencies>
  <dependency>
    <groupId>dev.elide</groupId>
    <artifactId>elide-core</artifactId>
    <version>${elide.version}</version>
  </dependency>
  
  <dependency>
    <groupId>dev.elide</groupId>
    <artifactId>elide-server</artifactId>
    <version>${elide.version}</version>
  </dependency>
</dependencies>
```
