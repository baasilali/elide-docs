# Child Process

API support and documentation for the `node:child_process` module.

**Module:** `node:child_process`  
**Support:** Alpha  
**Docs:** <a href="https://nodejs.org/api/child_process.html" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Node.js Child Process Docs</a>

## Import

```javascript
// ESM
import proc from "node:child_process"
```

```javascript
// CJS
const proc = require("node:child_process")
```
