## Highlights

**Elide is a runtime, like Node or Python.**

You install it on your machine as a standalone binary. Elide is statically linked and has essentially zero runtime dependencies.

You can install it on macOS or Linux with this one-liner:

```bash
curl -sSL --tlsv1.2 elide.sh | bash -s -
```

<br />

**Elide can run multiple languages.**

JavaScript, TypeScript, WASM, and Python are built-in. Ruby, JVM (Java, Kotlin, Scala), and LLVM (Swift, C++, Rust) are on the way.

```typescript
// hello.ts
import py from "./my-app.py"

// this line exists to show that this is typescript
const msg: () => string = () => `${py.say_hello()} + TypeScript!`
console.log(JSON.stringify({greeting: msg()}))
```

```python
def say_hello():
  """Render a greeting."""
  return f"Hello from Python"
```

```bash
elide ./hello.ts
```

```json
{"greeting": "Hello from Python + TypeScript!"}
```

<br />

**Elide is extremely fast.**

- Elide runs your Python code **up to 3x faster than CPython**.
- Elide runs your TypeScript code **faster than Node can run JavaScript**.
- Elide runs your HTTP endpoints at up to **800k RPS**. That's [really fast](/docs/performance).

> **Note**  
> Elide is independently benchmarked by TechEmpower. [Latest results](https://www.techempower.com/benchmarks/#hw=ph&test=plaintext&section=data-r23:~:text=133-,elide,-4%2C089%2C078)

<br />

**Elide supports the APIs you already know and the tools you already love.**

- Elide is [WinterTC](https://wintertc.org) compatible and passes [Test262](https://github.com/tc39/test262)
- Like other JS runtimes, Elide **supports a large slice of the [Node API](/docs/javascript-node-api)**.
- Works with **NPM and PyPI**, **CJS** and **ESM**.
- Insanely fast **dependency installation** (via [orogene](https://orogene.dev/) and [uv](https://github.com/astral-sh/uv)).
- Elide supports embedded [SQLite](/docs/javascript-sqlite).

<br />

**Elide provides strong security boundaries and memory safety by default.**

- Written in memory-safe [Kotlin](https://kotlinlang.org) and [Rust](https://rustlang.org).
- Strong filesystem and environment isolation.
- Virtualized filesystem support (mount a tarball as your fs!).
- Extensively tested: over 10,000 unit & conformance tests, in addition to [Test262](https://github.com/tc39/test262).
