# CLI References

Elide supports multiple sub-commands, similar to Git. This reference guide covers all available commands and options.

## Usage Syntax

```text
Usage:
  elide srcfile.{js|ts|jsx|tsx|py|kts...} [OPTIONS] [--] [ARG...]
  or:  elide {script} [OPTIONS] [--] [ARG...]
  or:  elide build|test|check|project [OPTIONS] [TASKS...]
  or:  elide info|help|discord|bug... [OPTIONS]
  or:  elide js|node [OPTIONS] [--code CODE] [FILE] [ARG...]
  or:  elide run|repl|serve [OPTIONS] [--code CODE] [FILE] [ARG...]
  or:  elide javac|kotlinc|jar|javadoc|... [OPTIONS] [SOURCES...]
```

Manage, configure, and run polyglot applications with Elide.

## Runner Commands

**`run`, `r`, `repl`**

Run a script file or a snippet of code; if no code is given, start an interactive REPL session. Elide will select an appropriate language if given a clearly identifiable source file (Python for `.py`, Ruby for `.rb`, etc.).

```shell
$ elide run app.js
$ elide run script.py
$ elide repl
$ elide run --code "console.log('Hello!')"
```

**`serve`**

Alias to `run` a server in any language; Elide will block until the server exits.

```shell
$ elide serve server.js
```

**`test`**

Run a polyglot script, server, or interactive shell in test mode.

```shell
$ elide test
$ elide test pattern
```

## Language-Specific Commands

**`js`, `node`**

Runs a JavaScript file. Equivalent to `elide run --javascript ...`. Note that JavaScript is the default language engine.

```shell
$ elide js app.js
$ elide node app.js
$ elide js --code "console.log('Hello!')"
```

**`python`**

Alias to `run` some Python. Equivalent to `elide run --python ...`.

```shell
$ elide python script.py
```

**`ruby`**

Alias to `run` some Ruby. Equivalent to `elide run --ruby ...`.

```shell
$ elide ruby app.rb
```

## Project Commands

**`project`**

Manage Elide projects, which are defined by `elide.pkl` or other project manifests.

```shell
$ elide project
```

**`init`**

Initialize a new Elide project.

```shell
$ elide init
```

**`build`**

For this or a specified project, run the build, or a script mapped at the name 'build' within `elide.pkl`, or project manifests like `package.json`.

```shell
$ elide build
$ elide build --native
```

**`dev`**

Run an Elide project in development mode.

```shell
$ elide dev
```

**`install`, `i`**

For this or a specified project, resolve and install all dependencies. Dependencies are installed for all declared ecosystems, modulo provided flags.

```shell
$ elide install
$ elide i
```

**`add`**

For this or a specified project, add one or more dependencies, and then install them. Dependency coordinates are accepted in many forms.

```shell
$ elide add package-name
$ elide add org.example:library:1.0.0
```

## JVM Toolchain Commands

**`kotlinc`**

The Kotlin compiler (kotlinc) is a command-line tool that compiles Kotlin programs.

```shell
$ elide kotlinc -- [KOTLINC_OPTIONS] [SOURCES...]
```

**`javac`**

The Java compiler, supporting up to JDK 24.

```shell
$ elide javac -- [JAVAC_OPTIONS] [SOURCES...]
```

**`jar`**

Builds and manages Java Archive files (JARs).

```shell
$ elide jar -- [TOOL_OPTIONS] [FILES...]
```

**`javadoc`**

Renders Java code into documentation.

```shell
$ elide javadoc -- [TOOL_OPTIONS] [FILES...]
```

**`native-image`**

The Native Image compiler (native-image) is a command-line tool that compiles native binaries.

```shell
$ elide native-image [OPTIONS]
```

**`jib`**

Jib can build container images from JVM or native apps.

```shell
$ elide jib [OPTIONS]
```

## Utility Commands

**`info`**

Show info about the current app and environment.

```shell
$ elide info
```

Displays information about your Elide installation, including version, supported languages, and environment details.

**`which`**

Print the path to a tool binary.

```shell
$ elide which kotlinc
$ elide which javac
```

**`help`, `docs`, `bug`, `issue`, `feature`**

Report an issue or bug, find help for using Elide. Can search Elide's documentation at this version, and can open links to GitHub.

```shell
$ elide help
$ elide docs run
$ elide bug
$ elide issue
$ elide feature
```

**`discord`**

Open or show a link to join the Elide Discord server.

```shell
$ elide discord
```

**`completions`**

Generate bash/zsh completion script for Elide.

```shell
$ elide completions
```

**`pkl`**

Run the Pkl command-line tools.

```shell
$ elide pkl [command]
```

**`lsp`**

Run an LSP instance for an Elide project.

```shell
$ elide lsp
```

**`mcp`**

Run an MCP instance for an Elide project.

```shell
$ elide mcp
```

**`secrets`**

Manage secrets for the current app.

```shell
$ elide secrets
```
