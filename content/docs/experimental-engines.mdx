# Experimental Engines

Elide has experimental support for **JVM**, **LLVM**, and **WebAssembly.** These engines are still under construction upstream, as part of <a href="https://graalvm.org" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">GraalVM</a>.

| Engine | Implementation | Status |
|--------|----------------|--------|
| JVM | <a href="https://www.graalvm.org/latest/reference-manual/java-on-truffle/" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Espresso</a> | Experimental - Linux amd64 only. |
| LLVM | <a href="https://www.graalvm.org/latest/reference-manual/llvm/" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Sulong</a> | Experimental |
| WebAssembly | <a href="https://www.graalvm.org/latest/reference-manual/wasm/" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">GraalWasm</a> | Experimental |
| Pkl | <a href="https://github.com/apple/pkl" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Apple Pkl</a> | Experimental |

Several experimental engines support multiple dialects.

| Language | Engine | Status |
|----------|--------|--------|
| Java | JVM | Experimental - Linux amd64 only. Not yet documented. |
| Kotlin | JVM | Experimental - Linux amd64 only. Not yet documented. |
| Groovy | JVM | Not yet available. |
| Scala | JVM | Not yet available. |
| C/C++ | LLVM | Not yet available. |
| Rust | LLVM | Not yet available. |
| Swift | LLVM | Not yet available. |
| Bitcode | LLVM | Experimental - Linux amd64 only. Not yet documented. |
| WebAssembly | WASM | Experimental - Not yet documented. |
