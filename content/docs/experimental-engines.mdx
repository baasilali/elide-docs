# Experimental Engines

Explore experimental language engines currently in development for Elide.

‚ö†Ô∏è **Warning**: These engines are experimental and may have limited functionality, bugs, or breaking changes.

## Lua

### Status: Alpha

Basic Lua support is available:

```lua
-- hello.lua
print("Hello from Lua!")

function factorial(n)
  if n == 0 then
    return 1
  else
    return n * factorial(n - 1)
  end
end

print(factorial(5)) -- 120
```

Run it:

```bash
elide run --experimental hello.lua
```

### Limitations

- Limited standard library
- No C modules
- Performance not optimized

## R

### Status: Early Alpha

Statistical computing with R:

```r
# stats.r
x <- c(1, 2, 3, 4, 5)
mean(x)

# Data frames
df <- data.frame(
  name = c("Alice", "Bob"),
  age = c(30, 25)
)

print(df)
```

### Limitations

- Basic R functions only
- No CRAN packages yet
- Limited plotting support

## LLVM Bitcode

### Status: Experimental

Run LLVM bitcode directly:

```bash
# Compile to bitcode
clang -emit-llvm -c program.c -o program.bc

# Run in Elide
elide run --experimental program.bc
```

### Use Cases

- Running optimized C/C++ code
- Cross-platform binaries
- Performance-critical code

## Espresso (Java)

### Status: Preview

Run Java bytecode:

```java
// Hello.java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello from Java!");
    }
}
```

```bash
# Compile
javac Hello.java

# Run
elide run --experimental Hello.class
```

### Polyglot Java

```javascript
// Call Java from JavaScript
const ArrayList = Java.type('java.util.ArrayList')
const list = new ArrayList()
list.add("Hello")
list.add("World")

console.log(list.toString())
```

## Kotlin

### Status: Preview

Kotlin support through Java interop:

```kotlin
// hello.kt
fun main() {
    println("Hello from Kotlin!")
    
    val numbers = listOf(1, 2, 3, 4, 5)
    val doubled = numbers.map { it * 2 }
    println(doubled)
}
```

## Scala

### Status: Experimental

Basic Scala support:

```scala
// hello.scala
object Hello {
  def main(args: Array[String]): Unit = {
    println("Hello from Scala!")
    
    val numbers = List(1, 2, 3, 4, 5)
    val doubled = numbers.map(_ * 2)
    println(doubled)
  }
}
```

## GraalPython Extensions

### Status: Experimental

Advanced Python features:

```python
# Native performance with @numba-like JIT
from graalpy import jit

@jit
def compute(n):
    result = 0
    for i in range(n):
        result += i * i
    return result

print(compute(1000000))  # Fast!
```

## Enabling Experimental Features

### Command Line

```bash
# Enable specific engine
elide run --experimental --engine=lua script.lua

# Enable all experimental features
elide run --experimental-all script.js
```

### Configuration

```pkl
// elide.pkl
experimental {
  engines = List("lua", "r", "llvm")
  features = List("advanced-jit", "experimental-apis")
}
```
