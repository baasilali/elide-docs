## Upgrade Guide

### From beta9 to beta10

```bash
# Backup your project
git commit -am "Backup before upgrade"

# Update Elide
curl -fsSL https://elide.dev/install.sh | bash

# Test your application
elide test

# Update configuration if needed
# See breaking changes above
```

### Testing Before Upgrade

```bash
# Run in a Docker container
docker run -v $(pwd):/app elide/elide:beta10 elide run /app/server.js
```
