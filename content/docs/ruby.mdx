# Ruby

Elide can execute your Ruby applications, while integrating with other languages like JavaScript and Python.

## Language Engine

Elide executes Ruby using <a href="https://github.com/oracle/truffleruby" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">TruffleRuby</a> from Oracle.

| Language | Ruby |
|----------|------|
| Standard | <a href="https://en.wikipedia.org/wiki/Ruby_MRI" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">MRI</a> `3.2` |
| Maturity | Alpha |
| Engine | <a href="https://github.com/oracle/truffleruby" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">TruffleRuby</a> |
