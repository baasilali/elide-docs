# URL

API support and documentation for the `node:url` module.

**Module:** `node:url`  
**Support:** Alpha  
**Docs:** <a href="https://nodejs.org/api/url.html" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Node.js URL Docs</a>

## Import

```javascript
// ESM
import url from "node:url"
```

```javascript
// CJS
const url = require("node:url")
```
