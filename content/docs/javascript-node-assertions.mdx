# Assertions

API support and documentation for the `node:assert` module.

**Module:** `node:assert`  
**Support:** Beta  
**Docs:** <a href="https://nodejs.org/api/assert.html" style="color: rgb(168, 85, 247); text-decoration: underline; font-weight: 500;">Node.js Assertion Docs</a>

## Import

```javascript
// ESM
import assert from "node:assert"
```

```javascript
// CJS
const assert = require("node:assert")
```
